# Shiv Kailas Vadevale Website

A responsive restaurant website for Shiv Kailas Vadevale, designed to showcase menu items, business information, and customer interaction through a feedback form.

## Live Demo

https://anki-007.github.io/shiv-kailas-vadevale/


## Overview

This project is a static website built using HTML, CSS, and JavaScript. It represents a local food business offering items such as vada pav, South Indian breakfast, and beverages. The design focuses on simplicity, readability, and a smooth user experience across devices.

## Features

* Responsive layout for mobile, tablet, and desktop
* Navigation bar with smooth scrolling
* Hero section with branding and call-to-action buttons
* Categorized menu display
* About section with business details and timings
* Contact section with a feedback form
* Basic form validation using JavaScript
* Clean and modern user interface with animations


## Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Animations)
* JavaScript (DOM manipulation and event handling)

## How to Run Locally

1. Clone the repository:
   git clone https://github.com/anki-007/shiv-kailas-vadevale.git

2. Navigate into the project directory:
   cd shiv-kailas-vadevale

3. Open the index.html file in any web browser
